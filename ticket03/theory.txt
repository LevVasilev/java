1.Всё состоит из классов. Чтобы программа работала, в основной класс нужно добавить  public static void main(string[]  args).
package название_пакета - все классы находятся в пакетах
import java.util… - подключаем, если надо использовать классы из других пакетов.
public, private (есть доступ только внутри класса), protected (есть доступ в текущем классе, пакете или в производных классах).
2.Термин сортировка означает упорядочение элементов/значений в определенном порядке, то есть по возрастанию или по убыванию. В Arrays есть метод sort().
Пузырек (обмен) - попарно сравнивает соседние элементы и меняет местами.
Вставки - массив постепенно перебирается слева направо. При этом элемент сравнивается со всеми предыдущими элементами и размещается так, чтобы оказаться в подходящем месте среди ранее упорядоченных элементов.
Благодаря ООП возможно создавать универсальные методы для разных типов данных, используя дженерики и Object.
